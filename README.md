# Projet LiDAR to LEGO 
Ce projet propose une chaîne de traitement complète ("pipeline") permettant de convertir des nuages de points LiDAR (format .laz) en modèles 3D constitués de briques LEGO (format .ldr).

Conçu dans le cadre d'un Projet d'Initiation à la Recherche (ING2 - ENSG), le code est optimisé pour traiter les données LiDAR HD de l'IGN (France), tout en restant compatible avec le standard ASPRS.

L'objectif est de transformer une donnée brute géographique en une maquette physique ou virtuelle (lisible dans LDView, Stud.io), en résolvant les défis de solidité structurelle (ajout de piliers, coques) et de nettoyage de bruit.
